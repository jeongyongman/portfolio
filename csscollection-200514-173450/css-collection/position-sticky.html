포지션이 static 이나 relative 나 sticky 이면
그 안에 들어 있는 그 박스 안에서 변경이 일어나게 되고
포지션이 absolute 이면 들어있는 근접한 부모박스들 중에
static 이 아닌 박스에 위치변경이 일어나고
fixed 이면 들어 있는 박스랑 상관없이 그냥 viewport
즉 브라우저에서 보여주는
viewport 에서 포지션 변경이 일어난다

sticky는 이렇게 기존에 들어 있던 자리를 유지하다가
이렇게 스크롤링이 되면서 더 이상 보여지지 않는 시점에
이렇게 우리가 지정한 top left 포지션에 붙어 있고요
그 자리를 그대로 공간을 유지하고 있기 때문에
다른 요소들도 이렇게 그 자리에 그대로 있는 게 보이죠 

          .box2 {
            background-color: hotpink;
            position: sticky; /* 그 박스안에서 변경이 일어난다 */
            top: 20px;
            left: 20px;
          }

fixed 는 들어있는 부모와는 상관없이
이 viewport 안에서 포지션이 일어나기 때문에
이렇게 빠져나와서 여기 0,0 이 지정이 되는 걸 볼 수가 있어요 

            .box3 {
              background-color: blue;
              position: fixed;
              top: 20px;
              left: 20px;
            }
